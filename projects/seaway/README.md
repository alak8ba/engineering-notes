## Objectif de ces notes

Cette documentation vise à :

- expliquer les décisions techniques
- garder une trace des choix d’architecture
- structurer les connaissances autour du projet
- faciliter la compréhension du système

Ces notes sont écrites dans une logique d’ingénierie :
comprendre, expliquer et documenter les systèmes construits.
